# Random Task Generator

## Автор
Саблин Олег Игоревич

## Описание
Графическое приложение на Python (Tkinter), которое генерирует случайные задачи.

Программа позволяет:
- Генерировать случайные задачи
- Фильтровать задачи по категориям
- Добавлять собственные задачи
- Сохранять историю в JSON
- Загружать историю при запуске

## Категории
- Учёба
- Спорт
- Работа

## Использование
1. Запустите файл `main.py`
2. Выберите категорию
3. Нажмите кнопку «Сгенерировать задачу»
4. История появится в списке

## Проверка
- Пустые задачи не добавляются
- История автоматически сохраняется

## Технологии
- Python
- Tkinter
- JSON
- random
